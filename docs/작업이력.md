## 48. 수집 강화(AI 분류 정밀화) 구현 + 소급 재분류 (로컬 전용)

**지시:** 무관 사업(정보시스템 감리/컨설팅 외) 유입 → 강화방안 수립·보고(§47 문서 `기능검토_수집강화_AI분류_v1.0.md`) → **계획대로 구현**.

**구현(classify.ts + classify_backfill.mjs 동시):**

- **(B) Stage0 조건부 제외**: `SUSPECT_DOMAIN`(교과목·교육과정·서베이·설문·실태조사·연구용역·산출방법·지원사업 운영·홍보·행사) + 코어신호 없음 → 드롭.

- **(E) 2차 검증(adversarial)**: 코어 명시 아닌 적재건 재확인(`verifyKeep`/`verifyLLM`) → 구축/운영/교육 의심 시 보류 강등(`CLASSIFY_VERIFY`).
- 놓침 보강: 코어에 `정보화 기본계획·정보화계획` 추가(false negative 감소).

**(G) 소급 재분류 실측(dry-run→반영):** 미아카이브 177(전부 감리/컨설팅) 재평가 → **9건 제외**(게임 AX 운영·빅데이터 교과목·빅데이터 서베이·산출방법 조사·K-CURE 데이터구축·KIOST 고도화·중소기업 컨설팅지원·정보화 기본계획 등) → 정분류 168 유지(감리 19·컨설팅 149). 재판정 9건 **소프트 아카이브+분류해제**(복원 가능). 노출(마감전) = 감리 3·컨설팅 31.
